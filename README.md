## What Data This Scraper Extracts

| Field Name            | Field Description |
|-----------------------|-------------------|
| userId                | The unique ID of the user. |
| isBlueVerified        | Indicates if the user is verified with a blue checkmark. |
| createdAt             | The date and time when the user's account was created. |
| description           | The bio description of the user. |
| followersCount        | The total number of followers the user has. |
| favouritesCount       | The number of tweets the user has liked. |
| mediaCount            | The number of media (photos/videos) posted by the user. |
| statusesCount         | The number of tweets or statuses posted by the user. |
| profileImageUrlHttps  | The URL to the user's profile image. |
| username              | The username handle of the user (e.g., @elonmusk). |
